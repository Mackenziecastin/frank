HOW IT WORKS:
================================================================================

Every day at 12:30 PM EST:
   1. Script connects to: sftp://mfts.adt.com
   2. Downloads from: /Resi_New directory
   3. File pattern: ADT_Athena_DLY_Lead_CallData_Direct_Agnts_YYYYMMDD.csv
   4. Processes report using existing adt_pixel_firing.py logic
   5. Fires pixels for qualifying sales
